# Introdução à programação de Computadores
Este repositório conta com 4 projetos desenvolvidos na disciplina de Introdução à programação de computadores, 
onde foi desenvolvido um jogo de super trunfo, primeiro com o cadastro das cartas e depois o jogo em si, 
comparando elas em "duelos". Também foi desenvolvido um jogo de xadrez e um jogo de batalha naval.
